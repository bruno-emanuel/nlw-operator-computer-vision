## Sobre o Projeto

O Rockit Vision é um sistema que reconhece gestos das mãos em tempo real utilizando a webcam.

O objetivo foi entender na prática como funciona um pipeline completo de visão computacional, desde a captura de dados até a classificação com machine learning.

---

## Como Funciona

1. Captura da webcam  
2. Envio dos frames para o backend  
3. Detecção dos pontos da mão com MediaPipe  
4. Classificação do gesto com machine learning  
5. Retorno do resultado para exibição em tempo real  

---

## Tecnologias Utilizadas

- Python
- OpenCV
- MediaPipe
- Scikit-Learn
- WebSockets
